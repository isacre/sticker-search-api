# Backend — sticker-search

API FastAPI com [uv](https://docs.astral.sh/uv/).

## Setup

```bash
cp .env.example .env
make install
```

## Desenvolvimento

```bash
make dev
```

- App: http://localhost:8000
- Docs: http://localhost:8000/docs
- Health: http://localhost:8000/health

## Comandos

```bash
make help    # lista todos os targets
make test
make lint
make format
```
